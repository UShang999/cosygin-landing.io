Перевод в РГУ им. А.Н. Косыгина

Подробное руководство по переводу из других высших учебных заведений. Следуйте нашим рекомендациям для успешного зачисления.

Важное ограничение

У абитуриента должна быть академическая разница не больше 8-ми предметов по специальности, куда он переводится. В противном случае в переводе могут отказать.

Предварительный этап

Проверка вакантных мест

Узнайте о наличии вакантных бюджетных мест (если перевод планируется на бюджет) и общих вакантных местах. Информация доступна на сайте: rguk.ru → Абитуриенту → Перевод из другого ВУЗа → Вакантные места

Обязательно укажите при обращении

  • Направление и профиль подготовки
  • Форму обучения (очная/заочная/вечерняя)
  • Курс / семестр перевода

Основная процедура

Подготовка и подача документов

Комплект документов для приёмной комиссии

  • Копия паспорта (фото + прописка)
  • Копия СНИЛС
  • Справка о статусе студента (оригинал, дата — текущий месяц)
  • Справка о периоде обучения (оригинал, дата — текущий месяц)
  • Копии лицензии и аккредитации ВУЗа (заверенные, только для негосударственных ВУЗов)
  • Заявление по форме (скачать на сайте в разделе «Перевод из другого ВУЗа»)
  • Согласие на обработку персональных данных (скачать на сайте в разделе «Перевод из другого ВУЗа»)
